# YuanqiZhiKe Toolkit

A collection of free offline frontend tools by YuanqiZhiKe.

## Project Structure

```
yuanqi-toolkit/
├── index.html                  # Homepage: platform positioning & brand intro
├── tools.html                  # Tools catalog: categorized tool listings
├── privacy.html                # GDPR privacy policy
├── cookie.html                 # Cookie notice
├── license.html                # Source code license agreement
├── customize.html              # Custom development services
├── DEPLOYMENT.md               # Deployment instructions
├── README.md                   # This file
├── assets/
│   ├── css/
│   │   └── style.css           # Global stylesheet
│   └── js/
│       └── main.js             # Shared JavaScript
└── tools/
    └── article-rewrite/        # Offline Article Rewriter tool
        ├── index.html          # Tool detail page
        ├── tool-index.html     # Actual tool (included in ZIP)
        └── article-rewrite.zip # Downloadable ZIP package
```

## Features

- **100% Static**: No backend, no database, no server-side code
- **Fully Offline**: All tools run in the browser without internet
- **Privacy-First**: No personal data collection
- **Responsive Design**: Works on desktop, tablet, and mobile
- **Compliance Ready**: GDPR-compliant privacy policy and cookie notice

## Quick Start

1. Open `index.html` in your browser
2. Browse available tools
3. Click Download on any tool
4. Accept the license agreement
5. Unzip and open the tool locally
