How to Use

  1. Click Choose a video file and select a video.
  2. Set the Number of frames to extract (2-20).
  3. Optionally set Start time and End time to limit the range.
  4. Choose thumbnail scale (smaller = faster).
  5. Choose layout: Grid for a contact sheet, or ZIP for individual downloads.
  6. Click Extract Frames and download the result.

Frames are extracted at evenly-spaced intervals. Useful for video analysis, stop motion, GIF creation, and creating contact sheets. All processing is local.
