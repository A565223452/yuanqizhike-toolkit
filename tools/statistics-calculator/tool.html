How to Use

  1. Enter numbers separated by commas, spaces, or new lines.
  2. See mean, median, mode, range, min, max.
  3. Get variance, standard deviation (population and sample).
  4. View quartiles, IQR, outliers, skewness, coefficient of variation.

Population formulas divide by n; sample formulas divide by (n-1) for unbiased estimates. Outliers fall outside [Q1-1.5×IQR, Q3+1.5×IQR]. Skewness > 0 = right-skewed, < 0 = left-skewed.
