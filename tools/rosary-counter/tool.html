How to Use

  1. Select mystery set based on day or preference.
  2. Tap Pray Hail Mary for each Hail Mary (10 per decade).
  3. Counter auto-advances to next decade after 10 Hail Marys.
  4. Use Skip Decade to move forward manually.
  5. Complete all 5 decades to finish rosary.

Rosary has 5 decades (50 Hail Marys total). Each decade focuses on a mystery from Jesus/Marys life. Days: Joyful (Mon, Sat), Sorrowful (Tue, Fri), Glorious (Wed, Sun), Luminous (Thu).
