How to Use

  1. Add a row for each component (e.g. Assignments 30%, Midterm 20%, Final Exam 50%).
  2. Enter the Weight (%) of each component. Total should be 100%.
  3. Enter your Score (%) for completed components. Leave blank if not yet taken.
  4. The Current overall updates automatically based on completed work.
  5. Enter your Target Overall Grade (e.g. 90 for an A).
  6. Click Calculate Required Final Exam Score to see the minimum score you need on remaining components.
  7. Optional: switch the Grade Letter Reference to see +/- letter grades.
