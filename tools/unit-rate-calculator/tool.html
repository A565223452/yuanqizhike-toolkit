How to Use

  1. Add items with + Add item button.
  2. Enter each item's name, price, quantity, and unit.
  3. Click Compare to find best value.
  4. Review unit prices and savings.

Unit rate = price ÷ quantity. Lower = better value. Common units: oz, lb, g, kg, L, ml, ea (each). Use same units for fair comparison.
