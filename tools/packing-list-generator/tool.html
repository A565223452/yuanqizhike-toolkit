How to Use

  1. Choose trip type and duration.
  2. Set climate and traveler type.
  3. Check options (laundry, carry-on only).
  4. Click Generate List.

Customized packing list based on trip type, climate, and duration. With laundry, pack half as many clothes. Carry-on only: pack efficiently, no liquids >3.4oz. Print list and use checkboxes while packing. Always carry essentials (documents, meds) in personal item.
