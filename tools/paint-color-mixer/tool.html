How to Use

  1. Choose Color 1 and Color 2 using the color pickers.
  2. Set the ratio (percentage) for each color. Optional: add Color 3.
  3. Select mixing mode: Additive (light), Subtractive (paint), or Average.
  4. Click Mix Colors to see the resulting color.
  5. Click Copy Hex to copy the hex code.

Subtractive mode mimics real paint mixing (combining colors darkens the result). Additive mode mimics light mixing (combining colors lightens the result). Use ratios to control the proportion of each color in the mix.
