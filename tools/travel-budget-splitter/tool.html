How to Use

  1. Set number of travelers and enter their names.
  2. Add expenses with description, amount, and who paid.
  3. Click + Add Expense for more items.
  4. Click Calculate Split for balances and suggested transfers.

Each person pays an equal share. The tool calculates who owes whom and minimizes the number of transactions needed to settle up. Negative balance = owes money; positive = gets money back.
