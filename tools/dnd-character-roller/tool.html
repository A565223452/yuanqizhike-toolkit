How to Use

  1. Choose Roll method: 4d6 drop lowest (most common), 3d6 straight, point buy, or standard array.
  2. Optionally select Class and Race for automatic stat priority and racial bonuses.
  3. Set Level for proficiency bonus calculation.
  4. Click Roll new for fresh dice, or Apply modifiers to re-render.

Stats are auto-assigned to your class's primary abilities. Racial bonuses applied after. Point buy costs: 8=0, 9-12=1-4, 13=5, 14=7, 15=9.
