How to Use

  1. Paste your HTML into the input area.
  2. Set the indent size (default 2 spaces).
  3. Click Format HTML to beautify.
  4. Click Copy Result to copy the formatted code.

Uses the browser DOM parser to rebuild a clean tree. Void tags (br, img, input) are handled correctly. All processing happens locally.
