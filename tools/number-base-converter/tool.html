Number Base Converter

Convert between Binary, Octal, Decimal, Hexadecimal and any custom base (2-36). Local, no upload.

How to Use

  1. Type or paste a number in the input box at the top.
  2. Choose the source base from the dropdown (Binary, Octal, Decimal, Hexadecimal).
  3. Conversions to all common bases appear instantly below.
  4. Toggle Group digits to insert separators (spaces every 4 bits or 3 digits).
  5. Toggle Show prefix to display language prefixes like 0b, 0o, 0x.
  6. Enter a custom base (26) and click + Add custom output to see the value in that base.
  7. Click any Copy button on the right to copy that representation.
Common bases: Binary base 2 digits: 0-1 Octal base 8 digits: 0-7 Decimal base 10 digits: 0-9 Hexadecimal base 16 digits: 0-9 A-F Base 36 base 36 digits: 0-9 A-Z Max supported base: 36 (uses 0-9 then A-Z)
