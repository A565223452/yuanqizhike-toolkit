How to Use

  1. Enter Distance in miles and Elevation gain in feet.
  2. Set Max elevation (higher altitude = harder).
  3. Choose Terrain type (smooth trail vs scrambling).
  4. Set your Fitness level and Backpack weight.
  5. Click Calculate difficulty.

Uses Naismith rule (30 min/mile + 30 min/1000 ft elevation) adjusted for terrain, pack weight, and altitude. Score ranges from Easy to Extreme.
