How to Use

  1. Click Select video file and load a video.
  2. Set start time (in seconds) and duration.
  3. Choose width (smaller = smaller GIF).
  4. Select frame rate (10fps is good for most).
  5. Click Convert to GIF and wait.
  6. Preview and Download the GIF.

GIFs use 256 colors and produce larger files than MP4. Keep clips under 5 seconds. For longer animations, use video formats instead. All processing in browser.
