How to Use

  1. Select your tuning (Standard, Drop D, Open G, etc.).
  2. Click Play next to each string to hear the reference tone.
  3. Tune your string to match the tone by ear.
  4. Click again to stop the sound.

String numbers go from thickest (1 / lowest) to thinnest (6 / highest). Standard tuning: E2 A2 D3 G3 B3 E4. For best results, use headphones and tune in a quiet room. Pluck your string gently while tuning.
