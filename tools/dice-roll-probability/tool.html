How to Use

  1. Select dice type (d4 to d100).
  2. Set number of dice.
  3. Choose target type (exact, at least, at most).
  4. Enter target value.
  5. Click Calculate Odds.

For 2d6, most likely sum is 7 (16.67%). For 3d6, average is 10.5 (used in D&D ability scores). 1d20 has flat 5% per side.
