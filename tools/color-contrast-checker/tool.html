How to Use

  1. Use the color pickers or type a hex value for foreground and background.
  2. The contrast ratio updates live as you change colors.
  3. Check pass/fail badges for WCAG AA and AAA levels.
  4. Click Swap Colors to reverse foreground and background.

Follows WCAG 2.1 contrast formula. Normal text needs 4.5:1 (AA) or 7:1 (AAA). Large text (18pt+ or 14pt+ bold) needs 3:1 (AA) or 4.5:1 (AAA).
