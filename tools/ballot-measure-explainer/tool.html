How to Use

  1. Select measure type (initiative, referendum, bond, etc.).
  2. Set vote threshold needed to pass.
  3. Paste the ballot measure text.
  4. Click Analyze Measure for explanation.

Ballot language is often intentionally confusing. Read the official voter guide, check funding sources, and look up independent analysis. Vote YES to approve, NO to reject.
