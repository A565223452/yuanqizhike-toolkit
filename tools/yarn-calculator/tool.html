How to Use

  1. Select your project type (scarf, sweater, blanket, etc.).
  2. Choose yarn weight (lace to super bulky).
  3. Enter skein size (yards per skein, on the label).
  4. Set price per skein and safety margin.
  5. Get total yardage, skeins needed, and cost.

Estimates are based on average adult sizes with worsted yarn. Thinner yarn needs more yardage; thicker yarn needs less. Always buy extra (10-20%) for safety and buy all skeins from the same dye lot to ensure color consistency.
