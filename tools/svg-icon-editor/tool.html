How to Use

  1. Paste SVG code in left box.
  2. See live preview on right.
  3. Adjust colors and stroke with controls.
  4. Optimize to reduce file size.
  5. Download or copy the result.

SVG is vector format - scales without quality loss. Always include xmlns="http://www.w3.org/2000/svg". Use viewBox for responsive scaling. Inline SVG allows CSS styling. Use currentColor for theming. Optimize to reduce file size.
