How to Use

  1. Enter Party size and average party level.
  2. Click + Add monster to add creatures to the encounter.
  3. Set each monster's Challenge Rating (CR) and count.
  4. Click Calculate difficulty or it auto-updates.

Difficulty uses official 5e XP thresholds. Encounter multiplier increases with monster count - many weak monsters are deadlier than few strong ones. Aim for Medium-Hard for balanced fights.
