Unicode — Chinese Converter

Convert Chinese / any text to Unicode escape sequences and back. Local, no upload.
Char-by-char breakdown

How to Use

  1. Paste your text into the input box (Chinese, English, or mixed).
  2. Choose Mode:
    • Encode convert text to Unicode escapes (\u4f60\u597d).
    • Decode convert \uXXXX sequences back to readable text.
    • Auto detect let the tool guess based on content.
  3. Pick the Format: JS \uXXXX, HTML &#NNNN;, CSS \HHHH, or raw UTF-8 bytes.
  4. Toggle Keep ASCII as-is (only encode non-ASCII) and Encode spaces.
  5. Click Convert output and per-character breakdown appear instantly.
