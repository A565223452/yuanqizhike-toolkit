How to Use

  1. Upload an image (JPG, PNG, WebP)
  2. Adjust filters using sliders: brightness, contrast, saturation, hue, blur, opacity
  3. Or select a preset filter from the dropdown (Grayscale, Sepia, Vintage, etc.)
  4. Preview updates in real-time
  5. Click Download to save the filtered image as PNG
  6. Click Reset to clear all filters
