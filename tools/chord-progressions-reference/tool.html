How to Use

  1. Select a key (major or minor).
  2. Choose a genre filter (or All).
  3. Click Find progressions for chord patterns.
  4. See actual chords in the selected key plus famous examples.

Roman numerals: uppercase = major, lowercase = minor, dim = diminished. Nashville numbering system used by studio musicians. I = tonic, IV = subdominant, V = dominant.
