How to Use

  1. Enter a JSON schema or select a template (User, Product, Order, Address)
  2. Schema values are type hints: "email", "name", "number:18-65", "oneof:A,B,C", etc.
  3. Set the number of records to generate
  4. Click Generate to produce mock data
  5. Copy or download the result as JSON

Supported Types

  • uuid, name, email, phone
  • number:min-max, boolean, date, datetime
  • city, country, zipcode, street
  • company, product, price, url, ip, color
  • oneof:A,B,C -pick one from list
  • string:N -random string of N chars
  • Nested objects are supported
