How to Use

  1. Select dice type (D4, D6, D8, D10, D12, D20, D100, or custom).
  2. Choose number of dice to roll.
  3. Set modifier (bonus/penalty, e.g. +5 for STR mod).
  4. Click Roll to see results.

Standard RPG dice: D4 (tetrahedron), D6 (cube), D8 (octahedron), D10 (pentagonal trapezohedron), D12 (dodecahedron), D20 (icosahedron), D100 (percentile). D&D uses d20 for attacks/saves. Critical hit = natural 20, critical fail = natural 1. Math.random() provides pseudo-random numbers.
