How to Use

  1. Select one or more images to watermark
  2. Choose watermark type: Text or Image (Logo)
  3. For text: enter watermark text, font size, color, and opacity
  4. For image: upload a logo/image to use as watermark
  5. Select position (corners, center, or tile for repeating pattern)
  6. Click Apply Watermark to process all images
  7. Download individual results or click Download All
