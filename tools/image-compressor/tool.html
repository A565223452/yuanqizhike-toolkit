How to Use

  1. Upload an image (PNG, JPG, or WebP).
  2. Adjust Quality (10-100%) - lower means smaller file.
  3. Set Max width to downscale large images (default 1920px).
  4. Tick Keep original format or convert to JPEG.
  5. Click Compress to preview before/after, then Download.

For photos, 60-80% quality usually saves 50-70% with no visible loss. For PNG screenshots, try converting to WebP for better compression. All processing is local.
