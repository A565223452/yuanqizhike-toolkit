How to Use

  1. Set meditation duration in minutes (1-180).
  2. Choose bell sound: Tibetan bowl, Zen bell, wind chime, or silent.
  3. Optional: Set interval bell to ring every N minutes (0 = off).
  4. Set starting delay (seconds to settle before meditation begins).
  5. Click Begin Meditation. Bells ring at start, intervals, and end.

Bells are synthesized using the Web Audio API (no audio files needed). For best experience, use headphones and find a quiet place. The timer continues even if you switch tabs.
