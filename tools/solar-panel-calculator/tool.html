How to Use

  1. Enter your monthly electricity bill.
  2. Set cost per kWh (US avg: $0.15).
  3. Set sun hours/day (varies by region).
  4. Choose system size (typical: 6-10 kW).
  5. See savings, payback period, and ROI.
