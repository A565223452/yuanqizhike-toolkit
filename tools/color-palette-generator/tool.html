How to Use

  1. Pick base color or enter HEX.
  2. Choose palette type (complementary, analogous, etc.).
  3. Click color to copy HEX.
  4. Copy CSS variables.

Palette types: Complementary (opposite hues), Analogous (adjacent hues), Triadic (3 colors 120° apart), Tetradic (4 colors 90° apart), Monochromatic (same hue, different lightness), Shades (darker variants). Use 60-30-10 rule: 60% dominant, 30% secondary, 10% accent.
