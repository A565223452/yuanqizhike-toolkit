How to Use

  1. Pick a scheme type: complementary, analogous, triadic, etc.
  2. Drag the hue slider or click directly on the color wheel.
  3. View the generated palette with HEX values.
  4. Click any color card to copy its HEX code.

Color harmonies: complementary = opposite hues, analogous = adjacent hues, triadic = 120° apart, tetradic = rectangle, square = 90° apart, split-complementary = base + two adjacent to its complement.
