How to Use

  1. Select algorithm (HMAC-SHA256, HMAC-SHA512, MD5, SHA256)
  2. Enter secret key
  3. Enter parameters as JSON
  4. Choose signature string format (sorted, JSON, or custom)
  5. Click Generate Signature to get the hash
  6. Use the verify section to check a signature
