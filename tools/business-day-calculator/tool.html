How to Use

  1. Choose mode: Calculate between dates or Add/Subtract business days
  2. Set the start/end date or base date
  3. Select which days count as weekends (Saturday + Sunday is default)
  4. Add custom holidays (one per line, format: YYYY-MM-DD)
  5. Click Calculate to see business days count or resulting date

Use Cases

  • Project deadline calculation (excluding weekends)
  • SLA / contract day counting
  • Shipping/delivery date estimation
  • International work scheduling with different weekend days (e.g., Middle East: Fri-Sat)
