How to Use

  1. Enter survey title and optional description.
  2. Click + Text Question, + Multiple Choice, etc. to add questions.
  3. Edit each question text and options.
  4. Toggle Required for mandatory questions.
  5. Click Generate Survey HTML to download.

Output is standalone HTML with inline styles - no dependencies. Form posts to "#" - update the action attribute to your backend endpoint.
