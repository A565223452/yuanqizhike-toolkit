How to Use

  1. Edit each cron field (minute, hour, day, month, weekday).
  2. Or click a preset for common schedules.
  3. See the generated expression and plain-English description.
  4. View the next 5 execution times.

Cron format: minute hour day-of-month month day-of-week. Use * for "any", */N for "every N", 1-5 for range, 1,3,5 for list. Examples: "0 9 * * 1-5" = 9am weekdays. "*/15 * * * *" = every 15 minutes. "0 0 1 * *" = midnight on the 1st of each month.
