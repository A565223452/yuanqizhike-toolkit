How to Use

  1. Select Reptile species from the dropdown.
  2. Confirm adult length (auto-filled, editable).
  3. Choose Age (juvenile enclosures are smaller).
  4. Select Enclosure type (affects heat retention).
  5. Click Calculate enclosure.

These are MINIMUM sizes - bigger is always better. Snakes need enclosure length >= body length. Arboreal species need height. Tortoises need maximum floor space.
