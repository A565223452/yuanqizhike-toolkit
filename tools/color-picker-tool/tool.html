How to Use

  1. Use the color picker or type a HEX value.
  2. See HEX, RGB, HSL, HSV, and CMYK conversions.
  3. Click Copy next to any value to copy it.
  4. View complementary, analogous, triadic colors, tints, and shades.

HEX is web standard. RGB is for screens. HSL is human-friendly (hue/saturation/lightness). CMYK is for print. Complementary = opposite on color wheel. Analogous = adjacent hues.
