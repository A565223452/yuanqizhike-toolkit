How to Use

  1. Enter date and measurements (weight, chest, waist, hips, arms, thighs).
  2. Click Save Entry to store in browser.
  3. View latest stats with changes from first entry.
  4. See waist/hip ratio for cardiovascular health indicator.

Healthy waist/hip ratio: < 0.85 (women), < 0.90 (men). Higher ratios indicate higher cardiovascular risk. Data stored locally in your browser only.
