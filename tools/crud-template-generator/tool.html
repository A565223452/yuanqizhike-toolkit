How to Use

  1. Enter entity name (e.g., User, Product, Order)
  2. Define fields (one per line, format: name:type)
  3. Select language/framework (Express, Flask, Laravel, Spring Boot, Django)
  4. Choose database type (SQL or MongoDB)
  5. Click Generate Code to get the CRUD template
  6. Copy or download the generated code
