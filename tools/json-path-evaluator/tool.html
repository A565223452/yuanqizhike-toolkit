How to Use

  1. Paste or type JSON data in the textarea.
  2. Enter a JSONPath expression (starts with $).
  3. Click Evaluate or use the example buttons.
  4. View matched values with token breakdown.

Implements a subset of JSONPath: $ root, .key, [n] index, [-1] last, [*] wildcard, .. recursive, [?(@.field)] filters. Uses eval() for filter expressions — do not use with untrusted data.
