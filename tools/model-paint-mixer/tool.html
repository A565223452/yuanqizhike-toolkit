How to Use

  1. Choose base color and mix color (presets or custom hex).
  2. Set base ratio and mix ratio (e.g. 3:1 means 3 parts base, 1 part mix).
  3. Select thinner percentage (0% for basecoat, 50% for glaze, 75% for wash).
  4. Enter output volume in drops.
  5. Click Calculate Mix for color preview and recipe.

Colors mix in RGB space (additive). Real paint pigments may behave differently. Test mixes on a palette. Acrylics dry slightly darker; oils yellow over time. For Warhammer/scale models: Citadel, Vallejo, and Army Painter work with this tool.
