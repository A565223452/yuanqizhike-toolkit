How to Use

  1. Select applicant age (adult or child).
  2. Choose application type (new, renewal, lost).
  3. Select document (book, card, or both).
  4. Choose processing speed.
  5. Click Calculate Fees.

Fees current as of 2024. Passport book valid 10 years for adults, 5 years for children. Card only valid for land/sea travel to Canada/Mexico/Caribbean.
