How to Use

  1. Fill in company and client info.
  2. Set invoice #, dates, currency.
  3. Add line items with quantity and price.
  4. Set tax % and discount if needed.
  5. Click Generate — opens print preview; choose "Save as PDF".

Uses browser print-to-PDF (no server). For automated invoicing: use jsPDF + autotable, or pdf-lib. Ensure totals auto-calculate. Save invoices with sequential numbers for accounting compliance.
