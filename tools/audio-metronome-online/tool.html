How to Use

  1. Set BPM with slider or select tempo marking.
  2. Choose time signature (2/4, 3/4, 4/4, etc.).
  3. Click Start or press Space.
  4. Use Tap Tempo to detect BPM by tapping.

BPM = Beats Per Minute. Standard tempo range: 40-240. Accent on beat 1 (high pitch 1000Hz, others 600Hz). Tempo markings: Largo (slow), Andante (walking), Allegro (fast), Presto (very fast). Tap Tempo averages last 4 taps for accuracy.
