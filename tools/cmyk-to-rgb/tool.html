How to Use

  1. Enter C, M, Y, K values (0-100%).
  2. See instant RGB, HEX, HSL conversion.
  3. Copy CSS code for your project.

Formula: R = 255 × (1-C) × (1-K), G = 255 × (1-M) × (1-K), B = 255 × (1-Y) × (1-K). CMYK (Cyan, Magenta, Yellow, Key/black) is used in printing — colors are subtractive. RGB (Red, Green, Blue) is for screens — colors are additive. Always design in RGB for digital; convert to CMYK for print. Some vivid RGB colors cannot be reproduced in CMYK print (out of gamut).
