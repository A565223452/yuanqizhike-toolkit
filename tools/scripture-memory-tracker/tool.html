How to Use

  1. Add verses: enter reference, translation, and text.
  2. Click + Add Verse to save.
  3. Click Start Practice to review due verses.
  4. Try to recall, then click Reveal Verse.
  5. Mark Got it right or Need more practice.

Uses spaced repetition: New (daily) -> 3-day -> Weekly -> Bi-weekly -> Mastered. Correct answers advance the level; wrong answers reset. Daily practice is key to long-term retention.
