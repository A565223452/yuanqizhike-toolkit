How to Use

  1. Select fragrance type (Parfum, EDP, EDT, EDC).
  2. Adjust oil concentration if custom.
  3. Set total volume (ml).
  4. Choose optional additive (water, fixative).
  5. Click Calculate for mixing recipe.

Concentrations: Parfum 20-30% (richest), EDP 15-20%, EDT 5-15% (most common), EDC 2-5% (lightest). Aging is critical for proper scent blending.
