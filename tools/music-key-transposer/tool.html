How to Use

  1. Enter chords separated by spaces.
  2. Choose original and target keys.
  3. Optional: set capo position.
  4. Click Transpose.

Transposition shifts all notes by same interval. C→D = +2 semitones. Capo on fret 2 raises pitch 2 semitones, so play C shapes to sound as D. Common keys: C (no sharps/flats), G (1 sharp), D (2 sharps). Relative minor is 3 semitones below major.
