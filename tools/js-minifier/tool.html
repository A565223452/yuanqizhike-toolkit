How to Use

  1. Paste your JavaScript code into the JavaScript Input textarea
  2. Select options: Remove comments, Remove newlines, Mangle variables
  3. Click Minify to compress your code
  4. Click Beautify to format minified code (reverse operation)
  5. Click Copy Result to copy the output to clipboard

Note

This is a lightweight minifier using regex-based transformations. For production use, consider Terser or UglifyJS. This tool works entirely in your browser — no code is sent to any server.
