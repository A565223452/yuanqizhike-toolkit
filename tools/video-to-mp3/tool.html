How to Use

  1. Click Select video file and load a video.
  2. Set start time and end time (0 = full video).
  3. Choose audio quality and sample rate.
  4. Click Extract Audio and wait.
  5. Preview and Download the WAV file.

Output is WAV (uncompressed PCM) for full quality. To get MP3, convert WAV with Audacity or FFmpeg. Processing happens in browser - no upload.
