How to Use

  1. Enter function f(x) using standard math notation.
  2. Set point x, order, step h.
  3. Click Calculate.

Numerical derivatives use central difference (more accurate than forward). Smaller h = more accurate but more rounding error. Optimal h ≈ 0.0001 for double precision. f'(x) = slope, f''(x) = concavity, f'''(x) = rate of change of curvature. For exact symbolic derivatives, use Wolfram Alpha or SymPy.
