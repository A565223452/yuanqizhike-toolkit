How to Use

  1. Enter Q&A pairs separated by your chosen separator (default |||).
  2. One pair per line.
  3. Choose output format for your target provider.
  4. Add optional system message.
  5. Click Generate Dataset to convert to JSONL.
  6. Download .jsonl file for upload to fine-tuning API.

Recommended: 100+ examples for good results. Each example 50-500 tokens. Validate format before upload. Hold out 10-20% for evaluation set.
