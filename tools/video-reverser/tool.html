How to Use

  1. Click Select video file and load a video.
  2. Choose output speed (1x, slow-mo, fast).
  3. Set max duration (keep under 15 sec - high memory use).
  4. Click Reverse Video and wait for processing.
  5. Download the reversed WebM video.

Reversing captures all frames first, so longer videos use lots of RAM. Keep clips short. Output has no audio (forward audio cannot sync with reversed video). All processing is client-side.
