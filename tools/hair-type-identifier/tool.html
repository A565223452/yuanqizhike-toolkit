How to Use

  1. Identify curl pattern (1=straight, 2=wavy, 3=curly, 4=coily).
  2. Select sub-type (A=fine, B=medium, C=thick).
  3. Choose hair density and porosity.
  4. Get personalized hair care routine.

Hair typing: 1 (straight), 2 (wavy), 3 (curly), 4 (coily). Sub-types: A (fine), B (medium), C (thick). Porosity test: drop hair in water - floats = low, sinks slowly = normal, sinks fast = high. Density: how many hairs per square inch.
