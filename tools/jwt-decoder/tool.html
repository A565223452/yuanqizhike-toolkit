How to Use

  1. Paste a JWT (three Base64 parts separated by dots).
  2. See decoded Header (algorithm) and Payload (claims).
  3. View standard claims: iat (issued at), exp (expires), sub (subject).
  4. Check expiration status (valid / expired).

JWT = header.payload.signature, all Base64URL-encoded. This tool decodes locally only - your token never leaves the browser. Decoding is not verification: always verify signatures server-side using the secret/public key.
