How to Use

  1. Describe the main subject (person, place, object).
  2. Choose art style (photorealistic, anime, oil painting, etc).
  3. Set lighting, composition, and mood.
  4. Pick color palette and quality descriptors.
  5. Choose aspect ratio for output dimensions.
  6. Select target AI tool (Midjourney/DALL-E/SD).
  7. Click Generate then copy or use variations.

Subject comes first, then modifiers. Be specific: "a 30-year-old woman with red hair" > "a woman". For artist styles, add "by [artist name]".
