How to Use

  1. Build your SKU pattern by adding parts. Each part can be:
    • Prefix — fixed text (e.g. SHO).
    • Category — pick from a list (APP, ELEC, HOME).
    • Sequence — auto-incremented number.
    • Random — random alphanumeric string.
    • Date — current date in YYYYMMDD format.
  2. Set Quantity, Separator, Sequence start, and padding.
  3. Choose UPPERCASE or lowercase output, and whether to deduplicate.
  4. Click Generate SKUs — the list appears with copy buttons.
