How to Use

  1. Click Shuffle to reset and shuffle the deck.
  2. Click Draw card for a single card.
  3. Set count and click Draw for multiple cards.
  4. For 5+ cards, see poker hand analysis.

Standard deck: 52 cards, 4 suits (----, 13 ranks each (A, 2-10, J, Q, K). Probability of drawing any specific card: 1/52 = 1.92%. Fisher-Yates shuffle used (truly random). A perfect shuffle (8 times) ensures randomness.
