How to Use

  1. Type a habit name and choose Build or Break.
  2. Click Add to create the habit.
  3. Click Mark each day you complete it.
  4. Track your streaks and stats.

Build good habits by tracking daily. Aim for a 66-day streak to make it automatic. Start small (1-2 habits). Use the 2-day rule: never miss 2 days in a row. Habit data is saved in your browser locally.
