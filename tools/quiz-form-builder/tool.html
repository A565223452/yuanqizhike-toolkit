How to Use

  1. Enter quiz title.
  2. Add questions, one per line: Question|choice1|choice2*|choice3. Asterisk marks the correct answer.
  3. Choose options: show score, show correct answers, shuffle.
  4. Click Generate, then copy HTML (includes auto-grading script).

Tip: each line is one question. Use | to separate choices. Mark the correct choice with * (e.g., Paris*). The generated HTML includes JavaScript for auto-grading. Embed in any LMS or website.
