How to Use

  1. Upload an image file (PNG, JPG, GIF, WebP, SVG).
  2. View the Base64 data URI and preview.
  3. Copy as Data URI, CSS background, or HTML img tag.
  4. To decode, paste a Base64 data URI in the decode box.

All processing happens in your browser — no upload to server. Base64 increases file size by ~33%. Recommended for images under 4KB. For larger images, use CDN with HTTP/2 multiplexing instead.
