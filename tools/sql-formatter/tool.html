How to Use


    

  1. Paste your SQL query in the input area.
    2. 

  2. Choose indent style and options.
    3. 

  3. Click Format SQL to beautify.
    4. 

  4. Click Copy to copy the formatted result.
    5. 



The formatter adds newlines before major keywords (SELECT, FROM, WHERE, JOIN, etc.), normalizes spacing, and optionally uppercases keywords. Syntax highlighting shows keywords in cyan, strings in green, numbers in orange.
