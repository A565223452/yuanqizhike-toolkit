Email Extractor

Extract email addresses from any text or HTML. Deduplicate, validate, export. Local, no upload.
Paste content above and click Extract.

How to Use

  1. Paste any text or HTML containing emails into the input box.
  2. Choose Sort order and toggle Deduplicate / Validate format / Filter out common test emails (like test@, example@).
  3. Select the Separator used when copying/exporting.
  4. Click Extract — found emails appear in a list with a copy button each.
  5. Use Copy All or Download to export the full list.
