How to Use

  1. Add URLs one by one using the form, or paste a list into the Bulk add URLs box and click + Add Bulk.
  2. For each URL set Last modified (date), Change freq (always/hourly/daily/weekly/monthly/yearly/never), and Priority (0.0 - 1.0).
  3. Click Generate Sitemap �?a valid XML sitemap is produced in the output box with stats.
  4. Use Copy XML or Download sitemap.xml to export.
  5. Upload the file to your site root and submit it via Google Search Console.
