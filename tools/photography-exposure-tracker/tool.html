How to Use

  1. Set aperture (f-stop), shutter speed, and ISO.
  2. Add scene/lighting description and lens used.
  3. Click + Add Entry to log each exposure.
  4. Click Analyze for EV, equivalent exposures, and reference.

EV (Exposure Value) = log2(N^2/t) at ISO 100. Higher EV = brighter light. Sunny 16 rule: f/16 at 1/ISO = EV 15 for bright sun. Use EV to find equivalent exposure combinations.
