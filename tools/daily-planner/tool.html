How to Use

  1. Enter task name in the input.
  2. Set priority (high, medium, low).
  3. Optional: set a time for the task.
  4. Click "Add" to add the task.
  5. Check off completed tasks.

Productivity tips: Eat the frog (do hardest task first). Time block similar tasks. Take 5-min breaks every hour. Single-task (avoid multitasking). Review day before sleeping. Plan tomorrow tonight. Use priorities to focus on what matters.
