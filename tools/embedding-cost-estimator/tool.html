How to Use

  1. Enter text volume in characters or upload a .txt file.
  2. Choose embedding provider (OpenAI/Cohere/Voyage).
  3. Set number of API calls per period.
  4. Click Estimate Cost to see breakdown.

Token estimate uses 4 chars/token (English). For custom pricing, select "Custom" and enter your per-1M-token rate. Storage assumes float32 vectors.
