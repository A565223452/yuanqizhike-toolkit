How to Use

  1. Enter file names (one per line) in the textarea
  2. Select a rename pattern from the dropdown
  3. Configure pattern options (prefix text, find/replace, number settings, etc.)
  4. Click Preview Rename to see the before/after comparison
  5. Click Copy New Names to copy the renamed list to clipboard

Available Patterns

  • Add Prefix -prepend text to all filenames
  • Add Suffix -append text before the file extension
  • Find & Replace -replace text in filenames
  • Sequential Number -add numbered prefix (001_, 002_, ...)
  • Case conversion -lowercase, UPPERCASE, or Title Case
  • Space handling -remove spaces or convert to underscores
  • Date prefix -add current date as prefix

Note: This tool generates renamed file names for preview. To actually rename files on your computer, use the generated names with a script or file manager.
