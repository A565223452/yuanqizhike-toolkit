How to Use

  1. Paste your .env file content.
  2. Choose output format (JSON, YAML, Table).
  3. Copy or download result.

.env format: KEY=VALUE per line. Comments start with #. Values can be quoted. Types auto-detected (string, number, boolean, null). Common in Node.js (dotenv), Docker, CI/CD. Never commit real secrets to git.
