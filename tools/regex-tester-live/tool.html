How to Use

  1. Enter your regex pattern (without slashes) and flags (g, i, m, s, u, y).
  2. Type test string in the textarea.
  3. View highlighted matches and detailed match info with capture groups.
  4. Use Replace to substitute matches ($1, $2, $& for full match).

Uses JavaScript's RegExp engine. Flags: g=global, i=ignore case, m=multiline, s=dotall (. matches \n), u=unicode, y=sticky.
