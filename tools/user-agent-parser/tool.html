How to Use

  1. Paste a User-Agent string in the text area.
  2. Or click Use my UA to parse your current browser.
  3. Or click Try sample for example strings.
  4. Click Parse to identify browser, OS, device, and bot.

User-Agent strings identify the client software. Note: UA strings can be spoofed. Modern browsers may "freeze" UA to reduce fingerprinting. For production, consider Client Hints (Sec-CH-UA header) or feature detection instead.
