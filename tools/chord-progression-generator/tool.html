How to Use

  1. Choose key and mode (major/minor).
  2. Select style (pop, rock, jazz, etc.).
  3. Set number of chords.
  4. Generate progression with chord diagrams.

Common progressions: I-V-vi-IV (pop, "Don't Stop Believin'"), ii-V-I (jazz), I-IV-V (rock), 12-bar blues. Roman numerals: uppercase = major, lowercase = minor, ° = diminished. Diatonic chords use only scale notes. Add 7ths for color (Cmaj7, G7, Am7).
