How to Use

  1. Enter subject (what to generate).
  2. Choose preset style and quality boosters.
  3. Add lighting, camera, artists.
  4. Pick negative prompt preset (default recommended).
  5. Add custom negative tags or use quick buttons.
  6. Set steps, CFG, sampler, size.
  7. Click Generate then copy config or JSON.

SD uses comma-separated tags. Emphasize with (tag:1.5), reduce with (tag:0.5). DPM++ 2M Karras is the best all-around sampler. Negative prompts are essential.
