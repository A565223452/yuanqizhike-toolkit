How to Use

  1. Basic: enter radius (r) and height (h), click Calculate.
  2. Solve: select known property, enter value and other dimension.

Cone: V = (1/3)πr²h, l = -r²+h²), L = πrl, S = πr(r+l). Slant height connects apex to base edge.
