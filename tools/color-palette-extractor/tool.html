How to Use

  1. Upload an image or use the default sample.
  2. Adjust the number of colors slider.
  3. Click any color to copy its hex code.

The tool uses k-means clustering to find the dominant colors in your image. Useful for: matching brand colors to a photo, creating cohesive designs, finding inspiration from nature photography, or extracting website color schemes from mockups.
