How to Use

  1. Click Start to begin timing.
  2. Click Lap to record split times.
  3. Click Stop to pause, Reset to clear.

Keyboard shortcuts: Space = Start/Stop, L = Lap, R = Reset. The stopwatch uses requestAnimationFrame for smooth updates (60fps). Suitable for sports, cooking, presentations, productivity timer. For high-precision timing (scientific use), use dedicated hardware.
