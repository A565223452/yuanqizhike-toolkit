How to Use

  1. Choose system size (2x2 or 3x3).
  2. Enter coefficients a, b, [c] and constant d for each equation.
  3. Click Solve System.

Solves a·x + b·y = d (and 3x3 version) using Cramer's rule. Determinant must be non-zero for unique solution. If det = 0, system has no or infinitely many solutions.
