How to Use

  1. Enter quilt dimensions (width and length in inches).
  2. Set block size (finished, without seam allowance).
  3. Set sashing width (between blocks) and border width.
  4. Enter blocks per row and number of rows.
  5. Select fabric width (typically 42 inches for quilting cotton).
  6. Set binding strip width (typically 2.5 inches).
  7. Click Calculate Quilt Fabric for total requirements.

Quilting uses 1/4-inch seam allowances. Block fabric is split across 4 colors by default. Backing calculations assume wide (108 in) fabric when available, otherwise pieced. Always buy extra for pattern matching and mistakes.
