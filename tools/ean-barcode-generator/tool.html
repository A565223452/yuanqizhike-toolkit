How to Use

  1. Pick the Format: EAN-13 (most common, 13 digits), EAN-8 (short, 8 digits), or UPC-A (US/Canada, 12 digits).
  2. Choose Input my numbers to paste your own, or Auto-generate to create random ones.
  3. For auto mode, set Count and an optional prefix (e.g. 690 for China, 400-440 for Germany).
  4. For input mode, paste numbers in the textarea — one per line. The check digit is calculated automatically if missing.
  5. Click Generate — barcodes render as PNG. Click any barcode to download it individually.
