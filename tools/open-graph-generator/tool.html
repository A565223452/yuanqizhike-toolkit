How to Use

  1. Enter og:title (the headline shown when shared).
  2. Choose og:type (website, article, product, etc.).
  3. Write og:description (1-2 sentences).
  4. Add og:url and og:image (recommended 1200x630).
  5. Set Twitter card type and handle.
  6. Preview the social card and copy the tags.

Recommended image size: 1200x630 pixels, under 8MB. Use summary_large_image for the big card preview on Twitter. Test with Facebook Sharing Debugger and Twitter Card Validator before deploying.
