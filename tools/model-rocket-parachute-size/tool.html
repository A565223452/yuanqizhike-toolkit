How to Use

  1. Enter your rocket weight in grams (with parachute and recovery wadding).
  2. Set desired descent rate (4-5 m/s is typical).
  3. Select parachute type (flat circular is most common).
  4. Enter launch altitude (apogee height) for drift calculation.
  5. Set wind speed for drift estimate.
  6. Click Calculate Parachute for diameter and specs.

Target descent rate: 3-5 m/s (10-16 ft/s) for safe recovery. Lower = gentler landing but more drift. Higher = less drift but harder impact. Always follow NAR (National Association of Rocketry) safety code.
