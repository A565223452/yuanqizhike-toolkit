Batch Password Generator

Generate strong, random passwords in bulk. Custom charset, length, count. Local, no upload.
Click Generate to create passwords.

How to Use

  1. Set Password length (longer = stronger; 16+ recommended) and Number of passwords.
  2. Tick the character sets you want included: lowercase, uppercase, digits, symbols.
  3. Untick Include ambiguous to remove look-alike characters (O 0 l 1 I) -useful when passwords are read aloud.
  4. Optionally enable Use custom charset and type your own character pool, or type characters to exclude in the same field.
  5. Click Generate -each password is shown with a strength badge and a copy button.
  6. Use Copy All or Download for the full batch.
