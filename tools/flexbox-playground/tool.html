How to Use

  1. Adjust flex-direction (row/column/reverse).
  2. Set justify-content (main axis alignment).
  3. Set align-items (cross axis alignment).
  4. Toggle flex-wrap and align-content for multi-row layouts.
  5. Add gap and change number of items.
  6. Click Copy CSS to copy the code.

Flexbox is one-dimensional. justify-content aligns along main axis (set by flex-direction). align-items aligns along cross axis. align-content only applies when wrap is enabled with multiple lines.
