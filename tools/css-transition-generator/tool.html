How to Use

  1. Pick the CSS property to transition.
  2. Set duration (how long) and delay (when to start).
  3. Choose an easing function or customize cubic-bezier.
  4. Try presets: ease, back, elastic, bounce.
  5. Click the preview box to see the animation.

Cubic-bezier(x1, y1, x2, y2) defines a custom easing curve. X must be 0-1 (time); Y can exceed for overshoot effects. Elastic/bounce use negative or > 1 Y values.
