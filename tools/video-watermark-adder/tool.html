How to Use

  1. Load a video file.
  2. Choose watermark type: text or image.
  3. Set position, opacity, font size, color.
  4. Click Apply — records video with watermark overlaid.

Browser-side using MediaRecorder + canvas captureStream. For text: choose contrasting color and add stroke for readability. For images: PNG with transparency works best. Output is real-time (length = video length).
