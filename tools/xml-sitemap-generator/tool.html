How to Use

  1. Enter URLs (one per line).
  2. Set default priority and frequency.
  3. Optionally enable auto-priority (depth-aware).
  4. Click Generate and download.

Submit sitemap at Google Search Console. Place at example.com/sitemap.xml. Max 50,000 URLs / 50MB per file. Use sitemap index for larger sites. Priority is relative (0.0-1.0). changefreq is a hint. Only include canonical, indexable URLs.
