How to Use

  1. Select fast type (Ramadan, Lent, Daniel, Intermittent, or Custom).
  2. Set start date for the fast.
  3. Adjust daily fast start/end times if needed.
  4. Set duration in days.
  5. Click Track Fast for progress and countdown.

Ramadan: dawn to dusk (varies by location/date). Lent: 40 days before Easter. Daniel Fast: 21 days plant-based. Intermittent: 16 hours daily. Always consult religious authorities for exact fasting rules.
