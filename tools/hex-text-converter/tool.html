How to Use

  1. Select mode: Hex — Text or Text — Hex
  2. Paste hex string or text in the input area
  3. Choose separator format (None, Space, 0x prefix, Comma, Dash)
  4. Select encoding (UTF-8 supports international characters)
  5. Click Convert to see the result
  6. Use Note: Swap to quickly switch input/output
