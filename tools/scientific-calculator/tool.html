How to Use

  1. Click number and operator keys, or type directly using your keyboard.
  2. Use sin/cos/tan, log/ln, sqrt, ^ for powers, x! for factorial.
  3. Toggle DEG/RAD for trig functions (default degrees).
  4. Press = or Enter to evaluate. Use C to clear, to backspace.

Supports π (pi), e (Euler's number), scientific notation. Order of operations: parentheses first, then ^, ×/÷, +/- Factorial works on integers up to 170.
