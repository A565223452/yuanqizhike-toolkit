How to Use

  1. Upload an image (simple high-contrast images work best).
  2. Choose your fabric count (14 CT is most common for Aida).
  3. Set the maximum width in stitches (controls detail vs. size).
  4. Choose the number of colors (more colors = more detail but harder to stitch).
  5. Select output: Pattern grid (printable with symbols) or Color preview.
  6. Click Generate Pattern then Print Pattern if needed.

For best results, use simple images with clear subjects. Photos with many gradients will create complex patterns. Start with 10-20 colors for your first project.
