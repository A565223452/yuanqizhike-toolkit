How to Use

  1. Click Upload image to load a photo.
  2. Choose an aspect ratio preset (or set custom).
  3. Drag on the image to select crop area.
  4. Set output dimensions and format.
  5. Click Apply crop to preview and Download to save.

All processing in your browser — image never uploaded. Presets cover common social media sizes (1:1 Instagram, 16:9 YouTube, 9:16 Story, 4:5 Facebook, etc.).
