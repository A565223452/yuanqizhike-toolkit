How to Use

  1. Set optional Sitemap URL and Crawl-delay in the global settings.
  2. Add rules with the + Add Rule button. Each rule targets a User-agent and applies Allow/Disallow paths.
  3. Click Generate robots.txt — the file is shown in the textarea with a validation log below.
  4. Edit the textarea directly if needed — the validation re-runs on every change.
  5. Use the Test URL box to check whether a specific URL is allowed or blocked for a chosen bot.
