How to Use

  1. Type a password in the box (it stays on your device).
  2. See strength rating, entropy bits, and crack-time estimate.
  3. Review the security checklist with green/red indicators.
  4. Click Generate for a strong random 16-character password.
  5. Follow the recommendations to improve weak passwords.

All checks run locally in your browser. We never transmit, store, or log your password. Crack time assumes 10 billion guesses/second (offline attack on fast hash like MD5). Use a unique password per site and a password manager.
