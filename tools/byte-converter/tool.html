Byte / Data Size Converter

Convert between bit, byte, KB, MB, GB, TB, PB. Binary (1024) or decimal (1000) mode. Local, no upload.

How to Use

  1. Enter a value and pick the source unit (bit, Byte, KB, MB, GB, TB, PB, EB or binary-prefixed KiB / MiB / GiB / TiB / PiB).
  2. Choose Decimal (1000) or Binary (1024) mode. Decimal uses SI multipliers (1 KB = 1000 B); Binary uses IEC multipliers (1 KiB = 1024 B).
  3. Adjust Decimals for output precision and toggle Group thousands for readability.
  4. All converted values appear instantly in the list below; click Copy next to any row.
Decimal (SI) : 1 KB = 10³ B = 1,000 B Binary (IEC) : 1 KiB = 2¹⁰ B = 1,024 B 1 Byte = 8 bits
