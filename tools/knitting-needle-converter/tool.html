How to Use

  1. Choose system to convert from (US, metric mm, or UK).
  2. Enter value of the needle size.
  3. Click Convert to see equivalent sizes.
  4. Reference full size chart below.

US sizes range from 0 (2mm) to 50 (25mm). UK/Canadian sizes are inverse: 14 is smallest, 0 is larger. Metric is most accurate for international patterns. Always check pattern gauge.
