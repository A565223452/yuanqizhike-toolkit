How to Use

  1. Define the assistant role clearly.
  2. Choose behavior (helpful, professional, concise, etc).
  3. Set response format (JSON, bullets, markdown, etc).
  4. List expertise areas the AI should draw on.
  5. Add specific rules and constraints.
  6. Set language and max length.
  7. Provide optional greeting and fallback.
  8. Click Generate then copy as text or JSON.

System messages shape AI behavior. Be specific, test with edge cases, iterate based on real usage. Keep under 1500 tokens for cost efficiency.
