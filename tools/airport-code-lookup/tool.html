How to Use

  1. Type search term: airport code (JFK), city (London), or airport name.
  2. Optional: filter by country.
  3. Choose sort order: code, city, name, or passenger volume.
  4. Click Search or just type - results update live.

IATA codes are 3-letter airport identifiers (e.g., JFK, LHR, DXB). ICAO codes are 4-letter (KJFK, EGLL). This tool covers major international airports worldwide. Passenger volumes are approximate annual totals.
