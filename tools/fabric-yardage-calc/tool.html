How to Use

  1. Select project type (dress, shirt, quilt, etc.).
  2. Choose size and fabric width.
  3. Enter project-specific details if prompted.
  4. Click Calculate Yardage for fabric amount.

Estimates are for solid fabrics. Patterns requiring matching may need 25% more. Prewash fabric before cutting to allow for shrinkage.
