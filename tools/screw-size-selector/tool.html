How to Use

  1. Select material being joined.
  2. Choose joint type (face, edge, pocket).
  3. Enter board thicknesses.
  4. Select load type.
  5. Click Find Screw Size.

Screw length should penetrate 2/3 of bottom board. Pilot hole prevents splitting. Hardwoods need slightly larger pilot holes than softwoods.
