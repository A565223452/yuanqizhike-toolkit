How to Use

  1. Select your state.
  2. Choose residency and duration.
  3. Add stamps/permits if needed.
  4. Click Estimate Cost.

Costs are approximate (2024). Resident: $10-60, Non-resident: $40-170. Add-ons: saltwater ($5-30), trout ($5-20), salmon ($5-25). Annual usually best value if fishing 5+ days. Seniors/kids/military often discounted. Verify on state wildlife agency website.
