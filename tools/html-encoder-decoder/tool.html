How to Use

  1. Enter text in the Input field.
  2. Click Encode to convert special chars to HTML entities.
  3. Or click Note: Decode to convert entities back to characters.
  4. Click Note: Swap to move output to input for chained operations.

HTML encoding converts <, >, &, ", ' to entities (&lt;, &gt;, &amp;, &quot;, &apos;) to display them as text instead of markup. Use when displaying code or user input on a webpage to prevent XSS attacks.
