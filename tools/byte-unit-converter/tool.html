How to Use

  1. Pick a Category: length, weight, temperature, volume, area, speed, time, or digital storage.
  2. Enter a value in the From field.
  3. Choose source and target units.
  4. Click Swap to reverse the conversion.
  5. See the result and a quick reference table for all units.

All conversions use SI base units. Temperature uses Celsius as the intermediate. Digital storage uses binary prefixes (1 KB = 1024 bytes).
