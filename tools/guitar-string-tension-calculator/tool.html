How to Use

  1. Select note you are tuning to.
  2. Choose scale length (Fender, Gibson, baritone).
  3. Select string gauge (thickness in inches).
  4. Click Calculate for tension in pounds.

Sweet spot: 15-25 lbs per string. For drop tunings use heavier gauge to maintain tension. Example: Drop C (C2) needs 0.060" or thicker to feel right.
