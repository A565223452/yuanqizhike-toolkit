How to Use

  1. Enter surface dimensions.
  2. Select wood type and stain type.
  3. Set coats and sides to cover.
  4. Click Calculate Stain.

Coverage varies by wood porosity. Softwoods absorb more than hardwoods. Transparent stains cover more than solid. Always test on scrap first.
