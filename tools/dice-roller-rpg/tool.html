How to Use

  1. Select a die type (D4, D6, D8, D10, D12, D20, D100, or D2 for coin flip).
  2. Set number of dice to roll.
  3. Add modifier (e.g., +5 for STR bonus).
  4. Click Roll or press Enter.

Standard D&D dice set: D4, D6, D8, D10, D12, D20, plus percentile D100 (or 2D10 for 1-100). Average roll = (sides + 1) / 2: D20 avg = 10.5. "Advantage" = roll 2d20, take higher (avg 13.8). "Disadvantage" = take lower (avg 7.2).
