How to Use

  1. Choose mode: Pixels — Print or Print — Pixels.
  2. Enter dimensions and DPI.
  3. See calculated print size or required pixels.

DPI (dots per inch) for print, PPI (pixels per inch) for screen — same calculation. 300 DPI = professional print quality. 150 DPI acceptable for posters. 72 DPI = web. Print size = pixels ÷ DPI. Megapixels = (W × H) ÷ 1,000,000. Common sizes: 4×6" needs 1200×1800px, A4 needs 2480×3508px at 300 DPI.
