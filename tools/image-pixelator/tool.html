How to Use

  1. Upload an image
  2. Choose mode: Whole image or Select region
  3. Adjust pixel size (2-50px) -larger means more pixelation
  4. For region mode: drag on the image to select an area
  5. Click Apply Pixelation to process
  6. Download the result as PNG

Use cases: Blur faces, license plates, sensitive text, or create retro pixel art effects.
