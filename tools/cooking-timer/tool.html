How to Use

  1. Set a custom timer or use a preset (egg, pasta, rice).
  2. Click to start, to pause.
  3. Click to reset a timer.
  4. Multiple timers can run simultaneously.
  5. An alarm sounds when time is up.

Keep this tab open while cooking. Timers run in your browser. The alarm uses your device speakers — make sure they're on. For long cooking, use a backup kitchen timer too.
