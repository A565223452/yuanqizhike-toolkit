How to Use

  1. Type search term: country name, code (+44), or ISO code (GB).
  2. Optional: filter by region (Europe, Asia, etc.).
  3. Click Search Codes to browse country codes.
  4. To format a number: enter it in phone number to format and click Format Phone.

Country calling codes are assigned by ITU. North America uses +1 (shared by US, Canada, Mexico, etc.). Most countries have unique codes. Trunk prefix (leading 0) is dropped when dialing internationally.
