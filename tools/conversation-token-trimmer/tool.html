How to Use

  1. Set system message (always preserved).
  2. Paste conversation as role: text lines.
  3. Choose max tokens and char/token ratio.
  4. Pick keep strategy: recent, summarize, or first+recent.
  5. Click Trim, copy the trimmed output.

Approximation: 1 token ~ 4 chars (English), 2 chars (code), 1.5 chars (CJK). For exact counts, use OpenAI tiktoken. Strategies: recent drops oldest, summarize adds summary of dropped, firstlast keeps first user message + recent.
