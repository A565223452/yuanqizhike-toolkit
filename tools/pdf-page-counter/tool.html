How to Use

  1. Click select or drag a PDF to the drop zone.
  2. View page count instantly.
  3. Process multiple files to see batch totals.

100% client-side — your PDFs never leave your device. Counts pages by parsing the PDF structure (/Type /Page entries). Works for most PDFs; some encrypted or unusual PDFs may report 0 pages — try unlocking first. For exact page count on protected PDFs, use a desktop tool.
