How to Use

  1. Select root note (C, C#, D, etc.).
  2. Choose chord type (major, minor, 7, etc.).
  3. Click Find Chord.

Chords built from intervals: Major = 0+4+7 (root, 3rd, 5th). Minor = 0+3+7. 7th chords add a 4th note. Standard tuning E-A-D-G-B-E. Fretboard shows chord tones highlighted. Common progressions: I-IV-V (blues), I-V-vi-IV (pop). Capo changes key without changing shapes.
