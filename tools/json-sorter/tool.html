JSON Data Sorter

Sort JSON arrays by any field with nested path support. Ascending / descending. Local, no upload.

How to Use

  1. Paste a JSON array of objects into the left input box.
  2. Enter the Sort by field for nested fields use dot notation, e.g. user.profile.age.
  3. Choose Order (ascending / descending) and Sort type (auto / number / string / date).
  4. Toggle extra options: Case-insensitive, Nulls last, Pretty print, Stable sort, Remove duplicates after sort.
  5. Click Sort the sorted array appears on the right with stats.
