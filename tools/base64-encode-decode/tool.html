How to Use

  1. Paste text or Base64 into the Input box.
  2. Click Encode to Base64 or Decode from Base64.
  3. Tick URL-safe to use - and _ instead of + and / (no padding).
  4. Use Use Output as Input to chain operations.

Base64 converts binary data to ASCII text using 64 characters (A-Z, a-z, 0-9, +, /). Used in data URLs, JWTs, email attachments, and basic auth. Output is ~33% larger than input. UTF-8 safe.
