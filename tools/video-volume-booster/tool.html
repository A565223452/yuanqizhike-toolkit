How to Use

  1. Click Choose a video file and select a video (must contain audio).
  2. Adjust the Volume boost slider (100% = original, 600% = 6x louder).
  3. Optional: Apply normalization for automatic level adjustment.
  4. Click Play with Boost to start playback with amplified audio.
  5. Watch the Audio Level Meter to monitor output level.

Uses Web Audio API for real-time amplification. Above 200% may cause distortion. For permanent volume change of a video file, use a video editor. Browsers require user interaction before audio playback.
