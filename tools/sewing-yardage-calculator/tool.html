How to Use

  1. Select fabric width (45, 54, 60, or 90 inches).
  2. Set seam allowance (typically 1/2 inch).
  3. Set hem allowance if needed.
  4. Enter pattern repeat if fabric has a repeating pattern.
  5. Add pattern pieces with length, width, and quantity.
  6. Click Calculate Yardage for total fabric needed.

The calculator optimizes piece layout across fabric width. Pattern repeat adds length for matching designs. 10% safety margin is included for errors. Always pre-wash fabric before cutting.
