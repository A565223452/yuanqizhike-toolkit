How to Use

  1. Set form title and choose fields.
  2. Enter action URL (Formspree, your backend, etc.).
  3. Pick color theme.
  4. Click Generate, then copy HTML.

For static sites without backend: Formspree (free 50/mo), Netlify Forms (100/mo free), Getform, Formspark. Always add Honeypot anti-spam field. For GDPR compliance: add consent checkbox, privacy link. WCAG: associate labels with inputs via for/id attributes.
