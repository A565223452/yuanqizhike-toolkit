How to Use

  1. Enter text, URL, or any content in the QR Code Content field
  2. Adjust size (64-1024px), error correction level, and colors
  3. Click Generate QR Code (or it auto-generates as you type)
  4. Download as PNG or SVG, or copy the data URL
  5. Use the quick type buttons for common QR code formats (URL, Email, Phone, SMS, WiFi, vCard)
