How to Use

  1. Choose QR type: URL, text, email, phone, SMS, WiFi, or vCard contact.
  2. Fill in the relevant fields that appear.
  3. Set Error correction level (higher = more damage tolerance).
  4. Choose size, margin, and colors.
  5. Click Generate QR Code then download as SVG or PNG.

QR codes are generated locally using the open-source qrcode-generator library (loaded from CDN once). Your data never leaves your browser.
