How to Use

  1. Enter Deck length and width in feet.
  2. Set Deck height (above ground - affects post requirements).
  3. Choose Decking material (wood, composite, PVC, or hardwood).
  4. Set Board spacing (1/8in for wood, 0 for composite) and Joist spacing.
  5. Click Calculate materials.

Composite and PVC need 12in joist spacing. Pressure-treated wood is most economical. Always check local building codes for railing and permit requirements.
