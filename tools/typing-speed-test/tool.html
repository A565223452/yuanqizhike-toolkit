How to Use

  1. Click Start Test to load a random passage.
  2. Type the text shown in the box as fast and accurately as you can.
  3. Watch live WPM and Accuracy updates.
  4. Test ends at 60 seconds or when you finish the passage.

WPM = words typed / minutes elapsed. Accuracy = correct characters / total characters. Correct characters are grey, errors are red-highlighted.
