How to Use

  1. Choose dimension (2D or 3D).
  2. Select operation.
  3. Enter vector components.
  4. Click Calculate.

Vector operations: magnitude (length), dot product (scalar, = 0 if perpendicular), cross product (3D only, perpendicular to both), angle (cos θ = A·B / |A||B|), projection (shadow of A on B). Applications: physics (force, velocity), graphics (normals), ML (cosine similarity).
