How to Use

  1. Enter a host name or IP address
  2. Use preset buttons or enter custom ports
  3. Set timeout (default 5s)
  4. Click Start Scan

Note: Browser-based scanning has limitations due to CORS. "Filtered" means the port may be open but cross-origin requests are blocked. For definitive results, use nmap.
