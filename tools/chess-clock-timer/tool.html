How to Use

  1. Choose a Time control preset (bullet, blitz, rapid, classical) or select Custom.
  2. For custom, set minutes and increment (seconds added after each move).
  3. Click White panel to start the clock for white.
  4. Click the active player's panel to switch turns.
  5. Use Pause to halt or Reset to restart.

Fischer increment adds time after each move. Common formats: Bullet 1+0, Blitz 3+2, Rapid 15+10, Classical 90+30. Click opposite panel to end your turn.
