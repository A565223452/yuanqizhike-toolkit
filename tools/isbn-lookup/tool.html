How to Use

  1. Enter ISBN (10 or 13 digits, hyphens optional)
  2. Click Validate Checksum to verify the ISBN is valid
  3. Click Look Up to fetch book info from Open Library
  4. View title, author, publisher, cover image, and more

Uses the free Open Library API for book data.
