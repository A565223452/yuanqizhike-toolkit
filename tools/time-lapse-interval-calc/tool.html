How to Use

  1. Select Scene type to auto-fill recommended interval and duration.
  2. Set desired Final video length in seconds.
  3. Enter Real-world duration of the captured event (use unit selector).
  4. Choose Frame rate (24, 30, or 60 fps).
  5. Optionally override Interval or leave blank for auto.

The 500 rule for stars: divide 500 by focal length to get max shutter speed (e.g. 500/24mm = 20.8 sec). Longer intervals = choppier motion. For smooth time-lapse, use shorter intervals for fast subjects.
