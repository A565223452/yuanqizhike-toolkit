How to Use

  1. Choose a field type from the dropdown.
  2. Enter the Field label and HTML name attribute.
  3. For select/radio fields, enter options comma-separated.
  4. Set required and placeholder if needed.
  5. Click + Add Field. Repeat for all fields.
  6. Preview appears live. Click Copy Code to copy the HTML.

Generated code is clean, semantic HTML5 with proper labels and accessibility attributes. Style it with your own CSS or any CSS framework.
