How to Use

  1. Type or paste text with emojis into the box.
  2. See the UTF-16 units (what most platforms count).
  3. See the code points (true character count).
  4. Get a breakdown of each emoji and its character cost.
  5. See the X weighted count for Twitter posts.

Emojis are complex: 😀 is 2 UTF-16 units, 👨‍👩‍👧 (family with ZWJ) is 11 units. Use this tool to plan posts for X (280 weighted), Instagram (2,200), and TikTok captions.
