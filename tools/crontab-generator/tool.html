How to Use

  1. Pick a preset or set each field manually.
  2. Enter the command to run.
  3. View the generated crontab line and human-readable description.
  4. Click Copy to copy to clipboard.

5-field standard cron format: minute hour day month weekday. Supports *, */N (step), lists (1,3,5), ranges (1-5), and L (last day of month). For advanced features (L for weekday, W workday, # nth weekday) consider using extended cron (Quartz).
