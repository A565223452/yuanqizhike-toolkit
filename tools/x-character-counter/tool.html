How to Use

  1. Type your X (Twitter) post in the text area.
  2. See the weighted character count in real time.
  3. URLs are counted as 23 characters regardless of length.
  4. Emojis and CJK (Chinese/Japanese/Korean) characters count as 2.
  5. Stay under 280 characters for a single post.

X uses a weighted character system. For long content, consider an X thread (up to 25 posts). Posts with 71-100 characters tend to get the highest engagement.
