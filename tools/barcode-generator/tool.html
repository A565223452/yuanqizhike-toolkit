How to Use

  1. Choose Barcode type: CODE-128 (any text), EAN-13 (13 digits), UPC-A (12 digits), Code 39 (alphanumeric), or EAN-8.
  2. Enter the data to encode.
  3. Set bar width, height, colors, margin, and text display.
  4. Click Generate Barcode then download as PNG, SVG, or JPEG.

EAN-13 and UPC-A require specific digit counts. Code 39 supports uppercase letters and digits only. CODE-128 supports all ASCII characters.
