How to Use

  1. Choose Operation (add, subtract, multiply, divide, or mixed).
  2. Select Difficulty level (1 = easiest, 4 = hardest).
  3. Set Number of problems and Columns.
  4. Click Generate.
  5. Click Show answers to display solutions.
  6. Click Print to print or save as PDF.

Great for elementary students, homeschoolers, and teachers. Multiplication stays within 12x12 even at higher levels for table practice.
