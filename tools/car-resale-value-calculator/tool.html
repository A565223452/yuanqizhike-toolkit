How to Use

  1. Enter purchase price and vehicle age.
  2. Enter current mileage and annual mileage.
  3. Select vehicle type, condition, and brand tier.
  4. Set project years ahead for future value.
  5. Click Calculate Value for depreciation analysis.

Cars lose 20-30% of value in first year. Trucks and exotics retain value best. EVs depreciate faster initially but slow after 3 years. Mileage over 12k/year reduces value by $0.10 per excess mile.
