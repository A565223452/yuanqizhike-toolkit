How to Use

  1. Select Page size (12x12, 8.5x11, 8x8, or 6x6 inches).
  2. Choose a Layout template (grid, collage, focal point, timeline, or scatter).
  3. Pick a Theme (birthday, wedding, vacation, baby, holiday, everyday).
  4. Enter a page Title.
  5. Click Generate for visual layout and materials list.

Each layout suggests photo arrangements, color schemes, materials, and theme-appropriate decorations. Print the layout as a reference guide while scrapbooking.
