How to Use

  1. Select HTTP method (GET, POST, PUT, DELETE, PATCH)
  2. Enter the API URL
  3. Add headers as JSON (e.g., {"Content-Type":"application/json"})
  4. For POST/PUT, enter request body
  5. Click Send Request to see response

Note: Some APIs block cross-origin (CORS) requests from browsers. If you get a CORS error, the API needs to enable CORS headers.
