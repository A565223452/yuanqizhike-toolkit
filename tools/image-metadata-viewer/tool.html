How to Use

  1. Click Choose file and select an image.
  2. See basic info (dimensions, file size, type).
  3. For JPEGs, view EXIF metadata (camera, settings, date).

EXIF data includes camera make/model, lens, focal length, aperture, shutter speed, ISO, date taken, and sometimes GPS coordinates. Note: most social media platforms strip EXIF data on upload. For privacy, use the EXIF Remover tool before sharing photos publicly.
