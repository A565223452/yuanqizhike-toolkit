How to Use

  1. Set Polish A name and color (use color picker).
  2. Set Polish B name and color.
  3. Adjust ratio slider (A:B percentage).
  4. Click Mix to see result and variations.

Mix on a palette or foil for accurate color. Start with light color as base, add dark gradually. Drops are approximate - mix thoroughly between additions.
