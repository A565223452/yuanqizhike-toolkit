How to Use

  1. Set number of dice and sides per die.
  2. Add modifier if needed.
  3. See statistics and probability distribution.
  4. Use probability calculator for specific outcomes.

Expected value of 1d6 = 3.5, 2d6 = 7, 3d6 = 10.5. Standard deviation grows with more dice. Distribution becomes more bell-shaped (normal) as dice count increases. Use probability calculator to find chance of rolling specific value or range.
