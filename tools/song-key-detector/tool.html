How to Use

  1. Enter chords separated by commas (e.g. C, G, Am, F).
  2. Click Detect Key.
  3. View detected key, scale notes, and related chords.

Supports sharps and flats. Common progressions: I-V-vi-IV (C-G-Am-F), ii-V-I (Dm7-G7-Cmaj7). For chord symbols like Cmaj7, Dm7, the base chord is used.
