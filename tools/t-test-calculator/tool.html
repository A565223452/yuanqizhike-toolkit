How to Use

  1. Select t-test type: one-sample, two-sample, or paired.
  2. Choose hypothesis (two-tailed or one-tailed).
  3. Enter data and set α.
  4. Click Calculate.

T-test compares means when σ is unknown. One-sample: compare group to a value. Two-sample: compare two groups (use Welch when variances unequal). Paired: before/after. Assumptions: normal data, independent observations. For n > 30, t ≈ z. Report: t(df) = value, p = value.
