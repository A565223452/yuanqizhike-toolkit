How to Use

  1. Enter input tokens per request (typical: 500-2000).

  5. Compare costs across all major LLMs.

Prices are per 1M tokens, as of mid-2024. Always check the vendor's pricing page for current rates. Most APIs bill separately for input (prompt) and output (completion) tokens. Output is typically 2-3x more expensive than input.
