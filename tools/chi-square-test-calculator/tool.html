How to Use

  1. Choose test type: goodness-of-fit or independence.
  2. For goodness-of-fit: enter "observed, expected" pairs.
  3. For independence: build contingency table, enter frequencies.
  4. Set α (significance level).
  5. Click Calculate.

Chi-square tests: Goodness-of-fit compares observed vs expected frequencies. Independence tests if two categorical variables are related. Assumptions: expected count ≥ 5 in each cell. If p < α, reject H₀ (no difference / no association).
