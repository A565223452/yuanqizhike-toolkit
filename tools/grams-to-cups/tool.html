How to Use

  1. Type grams or cups — the other updates.
  2. Select ingredient for accurate conversion (densities vary).
  3. See common amounts reference for the chosen ingredient.

Cup-to-gram conversions depend on ingredient density. 1 cup flour = 120g, but 1 cup sugar = 200g. Sifted flour weighs less (~100g/cup) than scooped (~120g). For consistent baking, use weight (grams), not volume. King Arthur Flour's website is a great reference for ingredient weights.
