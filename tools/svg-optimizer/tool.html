How to Use

  1. Paste your SVG code into the SVG Input textarea
  2. Select optimization options: remove comments, metadata, editor data, collapse whitespace
  3. Click Optimize SVG to process
  4. View the size savings in the info box
  5. Copy or download the optimized SVG
  6. Preview the optimized SVG rendering below

What gets removed

  • XML declarations and DOCTYPE
  • Comments (<!-- -->)
  • Metadata elements (Inkscape, Sodipodi, RDF, DC namespaces)
  • Editor-specific attributes (id, data-name, class)
  • Empty attributes and default values
  • Excessive whitespace and newlines
