How to Use

  1. Click number buttons or type on your keyboard.
  2. Use + − × ÷ for arithmetic, % for percentage.
  3. Press = or Enter to compute.
  4. C clears all, ⌫ deletes the last digit.

Keyboard shortcuts: 0-9, +, -, *, /, Enter (=), Backspace, Esc (clear). The calculator handles chained operations like 5 + 3 × 2 in left-to-right order.
