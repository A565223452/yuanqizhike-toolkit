How to Use

  1. Search by country name.
  2. Or filter by plug type to see which countries use it.
  3. View voltage, frequency, and plug types for each country.

Most modern electronics (laptops, phones) support 100-240V automatically — check the power brick. You only need a plug adapter, NOT a voltage converter. Hair dryers and curling irons may need a converter. Buy a universal adapter before traveling; airport ones are expensive.
