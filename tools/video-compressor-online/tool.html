How to Use

  1. Click Select video file and choose a video (MP4, WebM, MOV).
  2. Choose target width (lower = smaller file).
  3. Set video bitrate or use auto.
  4. Choose frame rate and audio bitrate.
  5. Click Compress Video and wait for processing.
  6. Preview and Download the compressed WebM file.

All processing happens in your browser - no upload. Output is WebM (VP9 codec). For MP4 output use a desktop tool like HandBrake. Larger videos take longer to process.
