JSON Field Cleaner / Dedup

Remove duplicates, null/empty values, flatten, sort keys. Local, no upload.

How to Use

  1. Paste your JSON (object or array of objects) into the left input.
  2. Tick the cleaning options you need:
    • Deduplicate identical objects in array — remove duplicate entries.
    • Deduplicate keys (keep last) — when an object has the same key twice, keep the last value.
    • Remove null / empty strings / empty arrays & objects — drop useless values.
    • Flatten nested objects — convert {a:{b:1}} → {"a.b":1}.
    • Sort keys / Trim strings / Pretty print — formatting options.
  3. Click Clean — the result appears on the right with before/after stats.
  4. Use Copy Result or Download to export the cleaned JSON.
