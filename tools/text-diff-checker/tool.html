How to Use

  1. Paste original text on the left, modified text on the right.
  2. Toggle options (ignore case, ignore whitespace).
  3. Click Compare to highlight differences.
  4. View stats, side-by-side, and inline diff.

Uses LCS (Longest Common Subsequence) algorithm to compute diff. For large inputs (> 5000 lines), performance may be slow. Compare at line-level; for char-level diff, use a dedicated diff tool.
