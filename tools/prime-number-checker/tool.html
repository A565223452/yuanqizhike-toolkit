How to Use

  1. Enter a number to check if it's prime.
  2. For composites, see prime factorization and all factors.
  3. For primes, see the next and previous primes.
  4. Use the second box to list all primes up to a limit.

A prime number has exactly two divisors: 1 and itself. 2 is the only even prime. Prime factorization expresses a number as a product of primes, e.g., 12 = 2² × 3.
