How to Use

  1. Enter the Original Price.
  2. Set one or more discounts (use + Add Discount to stack).
  3. Tick Add sales tax and enter the rate if needed.
  4. See savings, subtotal, tax, and final price.

Discounts apply sequentially (20% then 10% = 28% total, not 30%). Tax is applied to the discounted subtotal.
