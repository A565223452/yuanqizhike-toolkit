How to Use

  1. Set the quarter and team/owner name.
  2. Edit Objective titles and add new ones with "+ Add objective".
  3. For each objective, add Key Results (KRs) with current value, target value, and unit.
  4. Switch output format: Markdown, Plain text, or HTML.
  5. Click Copy OKR template to copy the output.

OKR best practices: 3-5 objectives per quarter, 3-5 KRs per objective, KRs must be measurable (use numbers/percentages). Avoid activity-based KRs—focus on outcomes. Review weekly, update values monthly.
