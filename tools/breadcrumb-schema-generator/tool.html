How to Use

  1. Enter your site base URL.
  2. Add breadcrumb items (name + path).
  3. Reorder or remove items as needed.
  4. Copy the JSON-LD schema into your page's <head>.
  5. Optionally copy the HTML breadcrumb markup for visual display.

BreadcrumbList schema helps Google show breadcrumb trails in search results. The last item should be the current page (no link in HTML, but URL is still required in schema).
