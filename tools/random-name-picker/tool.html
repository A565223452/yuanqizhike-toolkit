How to Use

  1. Type or paste names into the input box, one per line.
  2. Choose a mode: Pick 1 Winner, Pick N Winners, Split into Teams, or Shuffle Order.
  3. Set Count (for N winners) or Teams (number of groups).
  4. Toggle options: No repeats prevents picking the same name twice; Animate draw shows a spinning effect; Trim whitespace and Skip empty lines clean the input.
  5. Click 🎲 Draw to run the randomizer. Click again to redraw.
