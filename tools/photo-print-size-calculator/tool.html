How to Use

  1. Enter image dimensions in pixels.
  2. Select desired DPI quality.
  3. See print size and quality for standard sizes.

DPI guide: 300+ excellent, 200-300 good, 150-200 acceptable at distance, <150 pixelated. Print size = pixels / DPI. Standard sizes: 4×6", 5×7", 8×10", 8.5×11". For posters viewed from distance, lower DPI is acceptable. Always check effective DPI for each print size.
