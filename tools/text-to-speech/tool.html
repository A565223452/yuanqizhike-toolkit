How to Use

  1. Type or paste text into the input area.
  2. Select a voice from the available system voices.
  3. Adjust rate (0.5x - 2x) and pitch (0 - 2).
  4. Click Play to speak. Use Pause/Stop to control playback.

Uses the browser Web Speech API. Voices depend on your OS. Chrome, Edge, and Safari have the best support. No audio is uploaded - everything runs locally.
