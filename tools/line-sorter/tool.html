Multi-Line Text Sorter

Sort lines alphabetically, numerically, by length. Remove duplicates, shuffle, reverse. Local, no upload.

How to Use

  1. Paste your multi-line text in the input box on the left.
  2. Choose a sort mode from the dropdown (alphabetical, numerical, by length, reverse, shuffle).
  3. Tick any extra options: Remove duplicates, Trim each line, Remove empty lines, Natural sort, Sort by leading number.
  4. Click Sort (or just type — output updates live).
  5. Click Copy Result to copy the sorted output, or Download to save as a text file.
  6. Use ⇅ Swap to feed the output back as input for a second pass.
