How to Use

  1. Default durations: 25 min work / 5 min short break / 15 min long break. Customize as needed.
  2. Click Start to begin a work session. The tab title will show remaining time.
  3. When the timer ends, a sound plays and a browser notification appears.
  4. The timer auto-starts the next phase (break → work → break...).
  5. After every 4 work sessions (configurable), a long break is triggered.
  6. Use Pause / Reset / Skip as needed.
  7. Daily and total session counts are saved in your browser (per device).

Tip: Allow browser notifications when prompted, so you don't miss the end of a session when this tab is in the background.
