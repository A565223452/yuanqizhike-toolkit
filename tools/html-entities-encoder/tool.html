How to Use

  1. Paste text into the input box.
  2. Choose encoding type: Named (e.g. &amp;), Decimal (&#38;), Hex (&#x26;), or All.
  3. Click Decode to convert HTML entities back to characters.
  4. Use Swap to move output to input.

Encoding protects against XSS attacks when displaying user-generated content. Named entities are preferred for HTML special chars (< > & " '). Use decimal/hex for characters without named entities.
