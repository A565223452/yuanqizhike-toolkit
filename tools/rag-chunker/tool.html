How to Use

  1. Paste your text (article, document, transcript).
  2. Choose chunking strategy (chars/words/sentences/paragraphs/tokens).
  3. Set chunk size (200-1000 tokens typical for RAG).
  4. Set overlap (10-20% of chunk size recommended).
  5. Choose separator preference for clean breaks.
  6. Click Chunk Text to see results.
  7. Download as JSON or TXT.

For RAG: chunk size 200-500 tokens, overlap 10-20%. Smaller chunks = more precise retrieval. Break at semantic boundaries when possible.
