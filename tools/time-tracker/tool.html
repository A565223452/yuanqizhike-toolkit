How to Use

  1. Add a task with optional project name.
  2. Click ▶ to start tracking time.
  3. Click ⏹ to stop (saves the session).
  4. View total time and per-project breakdowns.

Tip: Track everything you do for a week to see where your time actually goes. Studies show people underestimate time spent on email/social media by 50%+. For billing, log sessions with notes in a separate doc. Data is not saved — refresh loses it.
