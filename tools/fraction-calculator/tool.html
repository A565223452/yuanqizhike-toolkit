How to Use

  1. Enter Fraction 1 numerator and denominator.
  2. Choose operation (add, subtract, multiply, divide).
  3. Enter Fraction 2.
  4. Click Calculate for result with steps.
  5. Use Simplify section for a single fraction.

Fractions are auto-simplified using GCD. Mixed numbers shown for improper fractions. Denominators cannot be zero.
