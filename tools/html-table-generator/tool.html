How to Use

  1. Set the number of rows and columns, then click Build Table.
  2. Tick Header row to make the first row use th tags.
  3. Edit cells directly in the table. Use + Row / + Col to expand.
  4. Click Export HTML or Export Markdown, then copy.

All editing is local. The table preview is fully editable. Markdown export adds the separator row automatically.
