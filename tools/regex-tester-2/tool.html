How to Use

  1. Enter regex pattern (without slashes).
  2. Set flags (g, i, m, s, u, y).
  3. Enter test string.
  4. Auto-tests on input change.

Common patterns: \d+ (numbers), \w+ (words), [a-z]+ (lowercase), ^pattern$ (full match), (group) (capture). Use g for all matches, i for case-insensitive. Test at regex101.com or here. JavaScript uses RegExp object.
